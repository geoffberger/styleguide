How to Use

This style guide is organized into sections that group together components used throughout the website. Each component includes the HTML necessary to add it to the website, which can be copy+pasted, and modified to your use. By default, this HTML is hidden from view and can be revealed using the "Toggle Code" button, either in the global nav or for each individual component.

Many components also include "notes", which provide further detail on how they can be used.
